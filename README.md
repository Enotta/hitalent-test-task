Консольное приложение на Python3 для работы с задачами.

Доступные операции:
- Создание
- Редактирование
- Фильтрация
- Поиск
- Смена статуса

Перед работой с тестами убедитесь что установлен pytest: 
```console
pip install --upgrade pytest
```
Для запуска тестов: 
```console
python -m pytest test_task_manager.py
```
